Restify
restify is a node.js module built 
specifically to enable you to build correct 
REST web services.
Why restify?
DTrace
KISS
5 
rules for writing 
good REST APIs
Nouns, no verbs
rpadovani.com/presentations
rpadovani.com/getpresentations
rpadovani.com/delpresentations
rpadovani.com/putpresentations

Two URLs per resource
HTTP GET (read)
GET rpadovani.com/presentations      - Retrieves all presentations
GET rpadovani.com/presentations/123  - Retrieves presentation with ID=123

HTTP POST (insert)
POST rpadovani.com/presentations     - Creates a new presentation
POST rpadovani.com/presentations/123 - Error
SSL Everywhere.
Period.
Use Let's Encrypt for free, signed certificates
Version via URL, not headers
rpadovani.com/api/v0/presentations
